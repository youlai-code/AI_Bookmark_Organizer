Chrome 内置 AI 需要最新版 Chrome (Canary/Dev) 并开启相关 Flags。无需 API Key，完全免费且隐私安全。

如何开启 Chrome 内置 AI:

  1. 请下载并安装 Chrome Canary 或 Dev 版本（标准版暂不支持）。
  2. 在地址栏输入 chrome://flags 并回车。
  3. 搜索 Optimization Guide On Device Model 并设为 Enabled BypassPerfRequirement。
  4. 搜索 Prompt API for Gemini Nano 并设为 Enabled。
  5. 重启浏览器，等待组件下载（可能需要几分钟）。
